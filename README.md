## 🎯 Цель проекта

Получить практические навыки командной разработки программного обеспечения с использованием Git, GitHub и базовых веб-технологий.

## ✨ Функциональность

- ✅ Добавление новых задач с указанием исполнителя
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Фильтрация задач (все/активные/выполненные)
- ✅ Счётчик общего количества задач и активных задач

## 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Git & GitHub

## 🚀 Как запустить проект

1. Клонируйте репозиторий:
git clone https://github.com/meschaninov/team-todo-list.git


2. Перейдите в папку проекта:
cd team-todo-list

3. Откройте файл index.html в браузере:
- **Windows**: дважды кликните на файл
- **macOS**: выполните `open index.html`
- **Linux**: выполните `xdg-open index.html`

## 📖 Как использовать
1. Введите текст задачи в первое поле
2. Укажите исполнителя во второе поле
3. Нажмите кнопку "Добавить"
4. Используйте чекбокс для отметки задачи как выполненной
5. Нажмите "Удалить" для удаления задачи
6. Используйте кнопки фильтров для отображения нужных задач
